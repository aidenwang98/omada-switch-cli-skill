## Chapter Notes

Official documentation states: OUI-Based VLAN (Organizationally Unique Identifier-Based Virtual Local Area Network) is a functionality that transmits and modifies the 802.p priority of packets within a specified VLAN based on the source MAC address of the packets. Similar to Voice VLAN, OUI-Based VLAN differs in the following ways: (1) OUI-Based VLAN does not provide users with default templates for OUI entries; users can configure them themselves; (2) Voice VLAN can only specify one VLAN as the Voice VLAN, whereas OUI-Based VLAN can assign different VLANs to different OUI entries; (3) The mask length for Voice VLAN is fixed, while OUI-Based VLAN allows for variable-length masks to be set for MAC addresses during configuration; (4) For each OUI entry, OUI-Based VLAN can specify which port the entry applies to.

- This chapter is marked `Only for Certain Devices`; treat all configuration commands as model-dependent.

## voice vlan

### Purpose
This command is used to configure OUI-Based VLAN entries. After the entries are configured, the system will automatically number each entry. Users can add ports when configuring entries, or add ports in port view after configuring entries. To delete the configured entries, please use the no voice-vlan command.

### Command Mode
Global Configuration Mode

### Syntax
```text
voice-vlan mac-addr mac-address oui-mask oui-mask vlan vlan-id priority priority oui-desc oui-description [interface {fastEthernet | gigabitEthernet | hundred-gigabitEthernet | port-channel | ten-gigabitEthernet | twentyFive-gigabitEthernet | two-gigabitEthernet} interface-value]
no voice-vlan index
```

### Parameters
```text
mac-address: MAC address, formatted as 00:0a:eb:00:13:86, represents the source MAC address that needs to be matched.
oui-mask: MAC address mask, formatted as ff:ff:ff:00:00:00, where the final match is the result of performing a bitwise AND operation between mac-address and oui-mask.
vlan-id: 802.1q VLAN ID, ranging from 2 to 4094.
priority: 802.1p priority, ranging from 0 to 7.
oui-description: Description information for the OUI entry, with a string length not exceeding 16 characters.
interface-value: Port number.
index: Automatically generated entry index number by the system.
```

### Default
Not explicitly specified in the official documentation.

### Example
```text
Add OUI-based VLAN entries:
Switch(config)#voice-vlan mac-addr 00:01:00:00:00:00 oui-mask ff:ff:00:00:00:00 vlan 100 priority 4 oui-desc oui1
```

### Notes
- Permission requirement: Only Admin, Operator and Power User level users have access to these commands.
- Availability: Only for certain devices according to the official chapter title.

## voice-vlan(interface)

### Purpose
This command is used to enable the OUI-Based VLAN function on the interface. To disable this function, please use the no voice-vlan entry command.

### Command Mode
Interface Configuration Mode

### Syntax
```text
voice-vlan entry index
no voice-vlan entry index
```

### Parameters
```text
index: The entry index number generated by the system is used to specify which OUI entry applies to a port through the index.
```

### Default
Not explicitly specified in the official documentation.

### Example
```text
Enable OUI-based VLAN on interfaces 1/0/2-6:
Switch(config)#interface range gigabitEthernet 1/0/2-6
Switch(config-if-range)#voice-vlan entry 1
```

### Notes
- Permission requirement: Only Admin, Operator and Power User level users have access to these commands.
- Availability: Only for certain devices according to the official chapter title.

## show voice-vlan

### Purpose
This command is used to view the OUI-Based VLAN configuration.

### Command Mode
Privileged EXEC Mode and Any Configuration Mode

### Syntax
```text
show voice-vlan
```

### Parameters
```text
None
```

### Default
Not explicitly specified in the official documentation.

### Example
```text
View the OUI-Based VLAN:
Switch# show voice-vlan
```
